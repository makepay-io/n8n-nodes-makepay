# Repository Protection

Recommended GitHub settings for `n8n-nodes-makepay`:

- Require pull requests before merging to `main`.
- Require one approval.
- Require status check `validate`.
- Require branches to be up to date before merging.
- Require conversation resolution.
- Enforce branch protection for administrators.
- Require linear history.
- Disable force pushes and branch deletions.
- Enable Dependabot alerts and security updates.
- Set Actions workflow token permissions to read-only.
